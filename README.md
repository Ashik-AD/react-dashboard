### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Ashik-AD/react-dashboard.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start local dev
   ```sh
   npm run dev
   ```
